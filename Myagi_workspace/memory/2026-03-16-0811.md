## 這個技能是什麼

**Exa** 是一個「語意搜尋引擎」的 MCP server。跟傳統關鍵字搜尋不同，你用自然語言描述想找的頁面，它幫你找到並直接回傳乾淨的內容（不需要另外爬）。

主要功能：
- `web_search_exa` — 語意網頁搜尋
- `web_search_advanced_exa` — 進階搜尋（可限定網域、日期、分類如論文/新聞/公司）
- `crawling_exa` — 給 URL 直接抓內容（含 JS 渲染頁、PDF）
- `company_research_exa` / `people_search_exa` — 公司/人物搜尋
- `deep_researcher_start` — 非同步深度研究報告
- `get_code_context_exa` — 搜程式碼範例

免費版有基本額度不需 API key，進階功能需要 key。
