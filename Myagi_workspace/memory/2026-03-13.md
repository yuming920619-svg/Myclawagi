## 11:01 UTC — MRAM ECC 參考論文清單
- source: chat
- tags: #mram #ecc #papers #references
- note:
  - 之後若再問「MRAM 目前主要使用哪種 ECC」可先回想這批論文與資料源。
  - Polar Coding for Spin-Torque Transfer Magnetic Random Access Memory (STT-MRAM), 2018, DOI: 10.1109/INTMAG.2018.8508795；文中明確提到 Everspin 16Mb MRAM 採 single-error-correcting (71,64) Hamming code。
  - Improving STT-MRAM Density Through Multibit Error Correction, 2014, DOI: 10.7873/DATE.2014.195（OpenAlex 也對到 10.5555/2616606.2616830）；主張 multibit ECC，重點偏 BCH。
  - Sanitizer: Mitigating the Impact of Expensive ECC Checks on STT-MRAM Based Main Memories, 2017, DOI: 10.1109/TC.2017.2779151；STT-MRAM 主記憶體需要 multi-bit ECC + scrubbing。
  - ORIENT: Organized interleaved ECCs for new STT-MRAM caches, 2018, DOI: 10.23919/DATE.2018.8342194；研究 cache 場景下 ECC bit selection。
  - An Adaptive Thermal-Aware ECC Scheme for Reliable STT-MRAM LLC Design, 2019, DOI: 10.1109/TVLSI.2019.2913207；依溫度動態調整 ECC 強度。
  - Efficient and Failure Aware ECC for STT-MRAM Cache Memory, 2019, DOI: 10.1007/978-981-13-5950-7_43；failure-aware ECC。
  - Fault-Aware ECC Scheme for Enhancing the Read Reliability of STT-MRAMs, 2023, DOI: 10.1109/ITC-Asia58802.2023.10301173；read reliability / failure-aware ECC。
  - Efficient BCH Code Encoding and Decoding Algorithm With Divisor-Distance-Based Polynomial Division for STT-MRAM, 2022, DOI: 10.1109/TMAG.2022.3142917；針對 STT-MRAM 最佳化 BCH 編解碼。
  - Small Area and High Throughput Error Correction Module of STT-MRAM for Object Recognition Systems, 2024, DOI: 10.1109/TII.2024.3362373；pipelined BCH module。
  - Proactive correction coset decoding scheme based on SEC-DED code for multibit asymmetric errors in STT-MRAM, 2018, DOI: 10.1016/j.mejo.2018.10.015；SEC-DED 延伸方案。
  - Channel Capacity and Soft-Decision Decoding of LDPC Codes for Spin-Torque Transfer Magnetic Random Access Memory (STT-MRAM), 2013, DOI: 10.1109/ICCNC.2013.6504145；LDPC 路線。
  - Design of Rate-Compatible Protograph LDPC Codes for Spin-Torque Transfer Magnetic Random Access Memory (STT-MRAM), 2019, DOI: 10.1109/ACCESS.2019.2959837；可調碼率 LDPC。
  - Recent Progress and Next Directions for Embedded MRAM Technology, 2019, DOI: 10.23919/VLSIT.2019.8776547 / 10.23919/VLSIC.2019.8777932；embedded MRAM 技術總覽。
  - Manufacturable 22nm FD-SOI Embedded MRAM Technology for Industrial-grade MCU and IOT Applications, 2019, DOI: 10.1109/IEDM19573.2019.8993454；eMRAM 產品化與 ECC-off 可靠性。
  - Reliability of 8Mbit Embedded-STT-MRAM in 28nm FDSOI Technology, 2019, DOI: 10.1109/IRPS.2019.8720429；embedded STT-MRAM 可靠性。
  - Overview of Reliability in Scaling Embedded STT-MRAM, 2025, DOI: 10.1109/IRPS48204.2025.10983522；縮放後 reliability 總覽。
  - 另有非論文但很關鍵的產品資料源：Everspin 16Mb MRAM Parallel Interface 頁面，寫明 internal single bit error correction code with 7 ECC parity bits for every 64 data bits。
