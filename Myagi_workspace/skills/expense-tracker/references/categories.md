# 消費分類對照表

## 預設分類

| 分類 | 關鍵詞範例 |
|------|-----------|
| 餐飲 | 早餐、午餐、晚餐、宵夜、便當、滷味、飯、麵、火鍋、壽司、漢堡、炸雞、小吃 |
| 飲料 | 咖啡、茶、奶茶、手搖、可樂、啤酒、果汁、水 |
| 交通 | 捷運、公車、Uber、計程車、加油、停車、火車、高鐵、YouBike |
| 日用品 | 衛生紙、洗衣精、牙膏、沐浴乳、洗髮精、口罩、電池 |
| 娛樂 | 電影、遊戲、KTV、書、漫畫、訂閱、Steam、Netflix |
| 學費 | 學費、教材、影印、列印、實驗耗材 |
| 服飾 | 衣服、褲子、鞋子、外套、帽子、襪子 |
| 醫療 | 看醫生、藥、掛號費、健保 |
| 其他 | 無法歸類的消費 |

## 分類規則

1. 若使用者明確指定分類（如「交通：捷運 35」），以使用者指定為準
2. 若未指定，根據品項關鍵詞自動判斷
3. 模糊不清時歸入「其他」，不要猜
4. 一句話包含多筆消費時，拆成多行分別記錄
